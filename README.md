# CSC 436 - Project 4: Node.js/Express.js Backend Fundamentals - Simple RESTful API

## Project Overview
This project is a simple RESTful API built with Node.js and Express.js.  
It manages a list of tasks and demonstrates full CRUD operations, JSON responses,
error handling, and testing with Postman.

## How to Run the API Locally

### Prerequisites
- Node.js and npm installed
- Postman installed
- Git installed

### Install Dependencies
```bash
npm install
